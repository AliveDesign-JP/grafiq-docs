## Description of Service

Grafiq is an application designed to help users create and manage personal graphs and charts. The App provides features including:

- Graph creation with 7 chart types (line, bar, pie, scatter, area, radar, horizontal bar)
- Manual data entry and tracking
- HealthKit data import (iPhone only, read-only)
- Technical indicators (SMA, EMA, trend lines, max/min)
- Scatter plot correlation analysis
- Multiple series per graph (Pro)
- iCloud sync across iPhone and Mac (Pro)
- Home screen widgets (Pro, iPhone only)
- Image export (Pro)
